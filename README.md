## Build

> [!WARNING]
> Despite being a public repository, it is not intended to be build locally.

Prerequisites:

1. Node.js
2. [Task](https://taskfule.dev) - task runner
3. Access to [submodules](.gitmodules)

```sh
task install
```

For fuerther instructions see:

```sh
task
```
